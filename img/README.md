# Static Assets

This directory contains static assets for the mimOE Developer Portal.

## Required Assets

Replace the placeholder files with actual mimik branding assets:

- `logo.svg` - mimik logo for navbar (recommended: 200x40px or similar aspect ratio)
- `favicon.ico` - Site favicon (recommended: 32x32px)
- `docusaurus-social-card.jpg` - Social media preview image (recommended: 1200x630px)

## Image Organization

Organize images by section:

```
img/
├── ai-foundation/
│   ├── quick-start-step1.png
│   └── quick-start-step2.png
├── platform/
│   └── architecture-diagram.svg
└── icons/
    └── checkmark.svg
```

Use PNG for screenshots and SVG for diagrams/icons when possible.
